公式庫
基本公式 ▾
平方關係
sin²θ + cos²θ = 1
商數關係
tan θ = sin θ / cos θ
互餘
sin(90°-θ) = cos θ
cos(90°-θ) = sin θ
導出
cos²θ = 1 / (1 + tan²θ)
和差角公式 ▾
sin
sin(A+B) = sinA cosB + cosA sinB
sin(A-B) = sinA cosB - cosA sinB
cos
cos(A+B) = cosA cosB - sinA sinB
cos(A-B) = cosA cosB + sinA sinB
tan
tan(A+B) = (tanA + tanB) / (1 - tanA·tanB)
tan(A-B) = (tanA - tanB) / (1 + tanA·tanB)
倍角公式 ▾
sin
sin(2θ) = 2 sinθ cosθ
cos（三種）
cos(2θ) = cos²θ - sin²θ
          = 2cos²θ - 1
          = 1 - 2sin²θ
tan
tan(2θ) = 2tanθ / (1 - tan²θ)
半角公式（進階） ▾
sin / cos
sin(θ/2) = ±√((1 - cosθ) / 2)
cos(θ/2) = ±√((1 + cosθ) / 2)
tan
tan(θ/2) = sinθ / (1 + cosθ)
          = (1 - cosθ) / sinθ
